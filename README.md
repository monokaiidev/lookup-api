## Getting Started

### Prerequisites

- **NodejS** (v18.x.x or later recommnded)
- **NPM** (v10.x.x or later recommended)
### Installation

1. **Clone the Repository:**
    ```bash
    git clone https://github.com/monokaiidev/lookup-api.git
    cd lookup-api
    ```

2. **Install Dependencies:**
    ```bash
    npm install
    ```

### Usage
1. **Run the Server**:
    ```bash
    node main.js
    ```
